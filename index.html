Jak czytać statusy w Dzienniku

🟢 Zdrowe (entry-friendly)

  • TP1 ✓ / TP2 ✓ — cele osiągnięte, partial profit lub close
  • SOS aktywny — wybicie nad TR top potwierdzone
  • Spring trzyma — formacja nadal ważna, oczekuje markup

🟡 Uwaga (czekaj)

  • ⚠ blisko SL — cena dryfuje w stronę stop loss
  • ⚠ testuje Spring — retest poziomu Spring low
  • ⚠ brak progresji — formacja stoi, brak higher-lows

🔴 NIE wchodź

  • ✗ Spring złamany — zamknięcie pod Spring low, formacja unieważniona
  • ✗ Stop loss — SL został dotknięty

📊 Tier-y (od najlepszego)

  • LPS — textbook Phase D, najwyższy winrate
  • CONFIRMED — Spring + bounce, jeszcze w TR
  • PHASE_D — post-SOS, brak LPS jeszcze
  • FRESH — Spring <3 dni, max niepewność
